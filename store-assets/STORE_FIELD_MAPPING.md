# Chrome 商店字段位置与填写指南

本文档总结了开发者后台 (Developer Dashboard) 填写字段与 Chrome 商店前台展示位置的对应关系，帮助你更准确地准备素材。

## 📋 字段映射表 (Field Mapping)

| 填写字段 (Field Name)                  | 必填  | 规格限制                                     | 商店展示位置 (Display Location)                                                      | 填写建议 & 作用                                                                                                                              |
| :------------------------------------- | :---: | :------------------------------------------- | :----------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------- |
| **应用名称**<br>(Name)                 |   ✅   | max 45 字符                                  | **1. 详情页顶部大标题**<br>**2. 搜索结果列表标题**<br>**3. 橱窗推荐标题**            | 最重要的字段。建议格式：`品牌名 - 核心功能简述`。<br>例如：`敲敲学英语 - 输出式英语学习助手`                                                 |
| **简短描述**<br>(Summary)              |   ✅   | max 132 字符                                 | **1. 标题下方** (详情页)<br>**2. 搜索结果摘要** (列表页)<br>**3. 推荐卡片摘要**      | 决定点击率的关键。用一句话讲清楚"它是做什么的" + "有什么好处"。<br>不要浪费在重复名字上。                                                    |
| **详细说明**<br>(Description)          |   ✅   | max 16,000 字符                              | **详情页 "概述" (Overview) 标签**                                                    | 支持 Markdown。详细介绍功能、使用场景、FAQ。这是用户决定是否安装的详细依据。<br>建议结构：痛点 -> 解决方案(本插件) -> 功能列表 -> 结尾号召。 |
| **商店图标**<br>(Store Icon)           |   ✅   | 128 x 128 px<br>PNG (无透明)                 | **1. 详情页标题左侧**<br>**2. 搜索结果列表左侧** (如果没传小图)<br>**3. 扩展管理页** | 品牌 Logo。务必清晰，不要包含太多文字。即使有其他尺寸，这个 128px 也是商店专用的。                                                           |
| **应用截图**<br>(Screenshots)          |   ✅   | 1280 x 800 px<br>or 640 x 400 px<br>JPEG/PNG | **详情页中部轮播图**                                                                 | **转化率核心**。第一张图最重要！<br>建议：<br>1️⃣ **功能图** (带文字标注)<br>2️⃣ **实景图** (使用场景)<br>3️⃣ **设置/高级功能图**                  |
| **小型宣传图**<br>(Small Promo Tile)   |   ❌   | 440 x 280 px<br>JPEG/PNG                     | **1. 搜索结果列表左侧**<br>**2. 相关推荐卡片**                                       | **强烈建议上传**。如果不传，Store 会强制裁剪你的 128px 图标，效果极差（留白过多）。此图能显著增加搜索点击率。                                |
| **大型宣传图**<br>(Marquee Promo Tile) |   ❌   | 1400 x 560 px<br>JPEG/PNG                    | **商店首页“精选推荐”轮播**                                                           | 如果你想被 Google 编辑推荐上首页，这是**必须**的。平时不显示，只有被推荐时才显示。                                                           |
| **类别**<br>(Category)                 |   ✅   | 下拉选择                                     | **标题下方的标签**                                                                   | 影响你在分类榜单中的位置。本项目选 `Productivity` (生产力) 和 `Education` (教育) 最合适。                                                    |
| **官方网址**<br>(Official URL)         |   ❌   | URL                                          | **详情页右侧边栏**                                                                   | 增加信任感。可以是你的 Landing Page 或 GitHub Pages。                                                                                        |
| **支持页面**<br>(Support URL)          |   ❌   | URL                                          | **详情页 "支持" (Support) 标签**                                                     | 用户点击"联系开发者"时跳转的地址。可以是 Github Issue 页或 Google Form。                                                                     |

## 🖼️ 视觉素材展示示意

### 1. 搜索结果页 (Search Result)
> 用户第一眼看到的界面

```text
+-------------------------------------------------------+
| [ 小型宣传图 (440x280) ]   应用名称 (Name)              |
|                        ⭐⭐⭐⭐⭐ (Rating)             |
| [  Logo + 品牌色   ]   简短描述 (Summary) |
| ----------------------------------------- |
| 类别 (Category)                           |
+-------------------------------------------------------+
```

### 2. 商品详情页 (Item Details)
> 用户决定安装的界面

```text
+-------------------------------------------------------+
| [图标]  应用名称 (Name)                                 |
| (128)  ⭐⭐⭐⭐⭐ | 用户数 | 类别                      |
|                                                       |
|  [ 简短描述 (Summary) ---------------------------- ]  |
|                                                       |
|  [ 安装按钮 (Add to Chrome) ]                          |
|                                                       |
|  +-------------------------------------------------+  |
|  |   截图轮播 (Screenshots 1280x800)               |  |
|  |   [图1] [图2] [图3] ...                         |  |
|  +-------------------------------------------------+  |
|                                                       |
| 概述 (Overview)        | 隐私 (Privacy) | 评价 (Reviews) |
| ---------------------- |
| 详细说明 (Description) |
| Markdown 内容区...     |
|                        |
+-------------------------------------------------------+
```

- [ ] **图片**:
    - [x] 图标 (128x128) - `icon-128.png`

    - [x] 小型宣传图 (440x280) - 已生成 V5/V6/V7 版本，选一个使用
